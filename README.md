# Edible Arcana - README

## Overview
Edible Arcana is a single-page restaurant website for a fictional Tarot-themed culinary experience. The website features a dark, mystical design with a fully responsive layout and interactive ordering form.

## Features
- **Themed Design**: Dark aesthetic with tarot-inspired color palette (#0e0b10 background, #e67e22 accents)
- **Responsive Layout**: Works on desktop and mobile devices
- **Interactive Menu**: Three signature tarot-inspired dishes with descriptions and pricing
- **Order Form**: Functional form with:
  - Quantity selectors for each dish
  - Add-on options
  - Side dish selection (radio buttons)
  - Special requests text area
- **Navigation**: Smooth scroll between Menu and Order sections

## Notes
- This is a front-end demonstration only
- Form submission does not connect to a backend
- Images are loaded from external CDN
- All styling is self-contained
